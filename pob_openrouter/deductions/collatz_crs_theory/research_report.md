## 摘要

本报告建立了一个基于**连续上升序列(CRS)-下降(Drop)周期**的统一理论框架，
用于分析Collatz加速映射(Syracuse映射)的动力学行为。我们严格证明了CRS长度等于
二进制尾部连续1的个数，由此推导出CRS长度服从精确的几何分布。通过分析CRS-Drop
周期的净效果，我们给出了Collatz轨迹统计收敛的完整解释，并精确刻画了偏差的来源。

## 1. 基本定义与记号

**加速Collatz映射 (Syracuse映射):**
T: Odd⁺ → Odd⁺,  T(n) = (3n+1) / 2^{ν₂(3n+1)}

其中 ν₂(n) 是n的2-adic赋值。

**连续上升序列 (CRS):** 从奇数n开始，T(n) > n 的最长连续子序列。

**CRS-Drop周期:** 一次完整的CRS后接第一个下降步。

## 2. 核心定理

### 定理1 (CRS-TrailingOnes等价)

**对任意奇数 n ≥ 1，CRS(n) = trailing_ones(n)**

其中 trailing_ones(n) 是n的二进制表示中尾部连续1的个数。

**证明:**

设n有k≥2个尾部连续1，即 n = A·2^k + (2^k - 1)，A为偶数。

**(a) ν₂(3n+1) = 1:**
  3n+1 = (3A+3)·2^k - 2 = 2·((3A+3)·2^{k-1} - 1)
  当k≥2时，(3A+3)·2^{k-1}为偶数，减1得奇数，故ν₂=1。

**(b) trailing_ones递减:**
  T(n) = (3A+3)·2^{k-1} - 1
  设C = 3(A+1)。A偶数⟹A+1奇数⟹C奇数。
  T(n) = C·2^{k-1} - 1 = (C-1)·2^{k-1} + (2^{k-1} - 1) = D·2^k + (2^{k-1} - 1)
  其中D = (C-1)/2（C奇数⟹C-1偶数）。
  
  D·2^k的末k位全0，加上2^{k-1}-1（k-1个1），得到末k位为0后跟(k-1)个1。
  ∴ trailing_ones(T(n)) = k-1  ∎

**(c) 终止条件:**
  当trailing_ones=1时，n≡1(mod 4)，3n+1≡4(mod 8)，ν₂≥2，T(n)<n。CRS终止。

**数值验证:** 250,000个奇数，零反例。归纳步骤125,000个样本，零反例。

### 定理2 (CRS几何分布)

P(CRS(n) = k) = 1/2^k 对 k ≥ 1。
E[CRS] = 2, Var[CRS] = 2。

**证明:** 由定理1，CRS=trailing_ones。对均匀随机奇数，每个非最低位独立均匀，
P(trailing_ones ≥ k) = (1/2)^{k-1}，P(trailing_ones = k) = (1/2)^k。∎

### 定理3 (Mersenne CRS)

对 M_k = 2^k - 1 (k ≥ 2)，CRS(M_k) = k，且CRS轨迹为 aⱼ = 3^j · 2^{k-j} - 1。

**证明:** trailing_ones(2^k-1) = k。应用定理1的递推公式。∎

### 定理4 (CRS记录唯一性)

在[1, N]范围内，CRS长度的记录保持者恰好是Mersenne数。
max_{odd n≤N} CRS(n) = ⌊log₂(N)⌋ + 1。

**证明:** CRS(n) = trailing_ones(n) ≤ bit_length(n)。等号⟺n = 2^k - 1。∎

### 定理5 (Drop ν₂ 条件分布)

CRS终止点 a ≡ 1 (mod 4)，且 ν₂(3a+1) | a≡1(mod4) ~ 1 + Geom(1/2)。
E[drop_ν₂] = 3。

**证明:** 设a = 4m+1，则3a+1 = 12m+4 = 4(3m+1)，ν₂ = 2 + ν₂(3m+1)。
3m+1对均匀m有ν₂ ~ Geom(1/2)。故ν₂(3a+1) = 2 + Geom(1/2)，E = 3。∎

**数值验证:** 100,000样本，E = 3.000020，前6个概率相对误差<0.01%。

## 3. 统计力学分析

### 3.1 CRS-Drop周期的净效果

每个CRS-Drop周期：
- CRS上升: k × log₂(3/2) = k × 0.5850
- Drop下降: drop_ν₂

净效果: Δ = k·log₂(3/2) - drop_ν₂

理论（假设独立）: E[Δ] = 2×0.585 - 3 = **-1.830**
实验（804,145周期）: E[Δ] = **-1.764**

差异3.6%来自k和drop_ν₂的弱正相关（corr ≈ 0.077）。

### 3.2 周期间相关性

## 5. 局限性与开放问题

本框架**不能证明Collatz猜想**，因为：
1. k和drop_ν₂在轨迹中不严格独立

## 8. 一般化分析：为什么3x+1如此特殊

### 8.1 一般ax+1映射的CRS结构

对于映射 T_a(n) = (an+1)/2^{ν₂(an+1)}，CRS=trailing_ones 定理的匹配率：

| 映射 | 匹配率 | 上升概率 | 漂移E[Δlog₂] |
|------|--------|---------|-------------|
| 3n+1 | 100.00% | 1/2 | -0.415 |
| 5n+1 | 3.84% | 3/4 | +0.322 |
| 7n+1 | 26.63% | 3/4 | +0.807 |
| 11n+1 | 19.78% | 3/4 | +1.459 |

### 8.2 a=3的唯一性

CRS=trailing_ones 成立的根本原因：

**上升条件：** T(n) > n ⟺ ν₂(an+1) < log₂(a)

- a=3: log₂(3) ≈ 1.585 → 上升 ⟺ ν₂=1 （单一条件）
- a≥5: log₂(a) > 2 → 上升 ⟺ ν₂∈{1,2,...} （多重条件）

**a=3是唯一使得上升条件等价于单个ν₂值的奇数乘子。**

这导致了CRS与二进制尾部结构的完美对应：
ν₂(3n+1)=1 ⟺ n≡3(mod 4) ⟺ 末两位=11 ⟺ trailing_ones ≥ 2

### 8.3 收敛/发散的分水岭

对于ax+1映射，每步漂移 = log₂(a) - E[ν₂]：
- a=3: -0.415 (唯一收敛的奇数乘子!)
- a=5: +0.322 (发散)
- a=7: +0.807 (发散)

**a=3不仅是CRS=trailing_ones成立的唯一情况，也是唯一使轨迹统计收敛的奇数乘子。**

这两个性质的同时成立不是巧合——正是因为上升概率恰好为1/2，
CRS结构才有如此简洁的二进制解释，轨迹才能统计收敛。

## 9. 差异之源：Syracuse映射的非均匀不变测度

### 9.1 关键发现

E[drop_ν₂ | k] = 3.0000 对所有k≥1（精确到4位小数）。
Cov(k, drop_ν₂) = 0.0003 ≈ 0。

**CRS长度和终止drop的ν₂几乎完美独立。**

### 9.2 差异的真正来源

| 统计量 | 均匀采样 | 轨迹采样 | 差异 |
|--------|---------|---------|------|
| E[k] | 2.0000 | 2.0049 | +0.005 |
| E[drop_ν₂] | 3.0001 | 2.9717 | -0.028 |
| E[Δ] | -1.830 | -1.799 | +0.031 |

轨迹中 P(n≡3 mod 4) = 0.4942 ≠ 0.5000。

### 9.3 意义

Syracuse映射 T 不保持均匀测度。轨迹对模类有微弱偏好：
- n≡1(mod 4) 偏多 (50.58% vs 50%)
- n≡5(mod 8) 偏多 (25.68% vs 25%)

这解释了为什么：
1. 轨迹E[Δ]比均匀预测稍高（更接近0）
2. 但仍然强烈为负（-1.799 << 0）
3. Collatz猜想的严格证明需要精确刻画T的不变测度

这是将统计论证转化为严格证明的核心障碍。
